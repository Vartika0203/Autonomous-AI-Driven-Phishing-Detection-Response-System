# Autonomous AI Phishing Detection System

This project implements an AI-driven phishing detection system
using Natural Language Processing (NLP) and Machine Learning.
The system autonomously detects and responds to phishing emails.

## Key Features
- NLP-based text preprocessing
- Machine learning classification
- Autonomous alert & response
- SOC-ready architecture

## Applications
- Email Security
- Cybercrime Investigation
- SOC Automation
